Store Inventory Management System (POS)

## Project Overview
A desktop application designed for local retail businesses (Sari-Sari Stores) to manage inventory and sales without internet access. Built using Python and SQLite.

## Features
* **Real-Time Inventory Tracking:** Automatically updates stock levels upon sale.
* **Database Management:** Uses SQLite for local, persistent data storage (no internet needed).
* **User Interface:** Clean, dark-mode GUI built with CustomTkinter for low-spec laptops.

## How it Works
1.  **Add Stock:** Input item name, price, and quantity.
2.  **Sell Item:** Enter the Item ID to instantly deduct stock and record the transaction.
3.  **Data Persistence:** Data remains saved even after closing the application.

## Tech Stack
* Python 3.12
* CustomTkinter (GUI)
* SQLite3 (Database)
